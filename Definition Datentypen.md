# Definition Datentypen

## Contactlist

### Status

- 0 : Anfrage geschickt
- 1 : Angenommen / Befreundet
- Ablehnen = löschen

## Guestlist

### Status

- 0 : Eingeladen
- 1 : Zugesagt
- 2 : Abgesagt

## Rating

### Value 

1-10 : dargestellt als Sterne

- 1 = 0,5 Sterne
- 2 = 1 Stern 
- ...
- 10 = 5 Sterne

## Task

### Status

- 0 : erstellt/offen
- 1 : erledigt

### User_ID

- NULL = kein User zugewiesen

## Todolistitem

### Status

- 0 : erstellt/offen
- 1 : erledigt

## User

### Gender

- NULL : keine Angabe
- 0 : männlich
- 1 : weiblich
- 2 : andere
